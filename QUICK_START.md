### Шаг 3: Подготовьте данные для обучения

Создайте структуру папок в `data/signatures/`:
```
data/signatures/
├── ivanov/
│   ├── sig_001.png

│   └── ... (5-10 подписей)
├── petrov/
│   ├── sig_001.png
│   └── ... (5-10 подписей)
```

### Шаг 4: Обучите модель
```bash
python ai/train_siamese.py --epochs 50 --batch_size 32
```

**Опции:**
- `--epochs 50` - количество эпох (рекомендуется 50-100)
- `--batch_size 32` - размер батча
- `--lr 0.001` - скорость обучения

Мониторинг в TensorBoard:
```bash

# Откройте http://localhost:6006
```

## 📊 Архитектура нейросети

**Siamese Network:**
- Два одинаковых энкодера с общими весами
- 4 сверточных блока (Conv2D + BatchNorm + ReLU + MaxPool + Dropout)
- Вход: 144x144 grayscale изображение
- Выход: 128-мерный вектор (эмбеддинг)
- Contrastive Loss для обучения парами

**Принцип работы:**
1. Обе подписи проходят через энкодер
2. Получаем два эмбеддинга
3. Вычисляем косинусное сходство
4. Если > 0.7 → подписи совпадают

## 💡 Советы

1. **Больше данных = лучше качество** (минимум 20 человек × 5 подписей)
2. **Обучайте 50-100 эпох** для хорошей точности
3. **Качественные сканы** подписей без шумов
4. **Аугментация включена** автоматически (повороты, сдвиги, шум)

## 🔧 Если что-то не работает

**"Модель не найдена"** → Запустите обучение сначала

**"Недостаточно данных"** → Нужно минимум 2 человека с 2+ подписями

**Низкая точность** → Добавьте больше данных, увеличьте эпохи

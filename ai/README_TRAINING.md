## Шаг 1: Подготовка данных

Создайте структуру папок для датасета:

```bash
mkdir -p data/signatures
```

Разместите подписи в папках по именам людей:

```
data/signatures/
├── person_001/
│   ├── sig_001.png
│   ├── sig_002.png
│   ├── sig_003.png
│   └── ...
├── person_002/
│   ├── sig_001.png
│   ├── sig_002.png
│   └── ...
└── ...
```

**Требования к данным:**
- Минимум 20 человек
- По 5-10 подписей от каждого человека
- Форматы: PNG, JPG, JPEG, BMP
- Изображения должны быть в оттенках серого или цветными (автоматически конвертируются)
- Рекомендуемый размер: от 200x200 пикселей

## Шаг 3: Обучение модели

Запустите обучение:

```bash
python ai/train_siamese.py --data_dir data/signatures --epochs 50 --batch_size 32
```

**Параметры:**
- `--data_dir`: путь к датасету (по умолчанию: data/signatures)
- `--epochs`: количество эпох (по умолчанию: 50, рекомендуется 50-100)
- `--batch_size`: размер батча (по умолчанию: 32)
- `--lr`: learning rate (по умолчанию: 0.001)
- `--margin`: margin для contrastive loss (по умолчанию: 1.0)
- `--embedding_dim`: размерность эмбеддинга (по умолчанию: 128)

**Примеры:**

```bash
# Быстрое тестирование (10 эпох)
python ai/train_siamese.py --epochs 10

# Полное обучение (100 эпох) на GPU
python ai/train_siamese.py --epochs 100 --batch_size 64

# С измененными параметрами
python ai/train_siamese.py --epochs 75 --lr 0.0005 --margin 1.5
```

## Архитектура сети

Siamese-сеть состоит из:
1. **Encoder** - сверточная сеть для извлечения признаков:
   - 4 блока Conv2D + BatchNorm + ReLU + MaxPool + Dropout
   - Вход: 144x144 grayscale изображение
   - Выход: 128-мерный вектор (эмбеддинг)

2. **Contrastive Loss** - функция потерь для обучения парами:
   - Сближает эмбеддинги одинаковых подписей
   - Отдаляет эмбеддинги разных подписей

## Советы по улучшению качества

1. **Больше данных**: чем больше подписей, тем лучше
2. **Аугментация**: включена автоматически (повороты, сдвиги, шум)
3. **Баланс классов**: скрипт автоматически создает сбалансированные пары
4. **Длительное обучение**: 50-100 эпох обычно достаточно
5. **Ранняя остановка**: модель сохраняет лучшие веса автоматически

## Troubleshooting

**Ошибка: "Модель не найдена"**
- Убедитесь, что обучение завершено успешно
- Проверьте путь к модели в signature_verifier.py

**Ошибка: "Недостаточно данных"**
- Нужно минимум 2 человека с 2+ подписями каждый
- Рекомендуется 20+ человек с 5+ подписями

**Низкая точность**
- Увеличьте количество эпох
- Добавьте больше данных
- Проверьте качество изображений

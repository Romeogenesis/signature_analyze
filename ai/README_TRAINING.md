# Инструкция по обучению и использованию Siamese сети для сравнения подписей

## 📋 Обзор

Этот проект использует **Siamese Neural Network** (двухканальную нейронную сеть) для сравнения рукописных подписей. Архитектура основана на предобученной модели ResNet18 с контрастивной функцией потерь.

## 🧠 Как работает Siamese сеть

### Архитектура

```
Подпись 1 ──┬──> ResNet18 (без последних слоёв) ──> Embedding (128D) ──┐
            │                                                          ├──> Distance ──> Similarity %
Подпись 2 ──┴──> ResNet18 (те же веса!) ───────> Embedding (128D) ──┘
```

**Ключевые особенности:**
- Обе ветви используют **одинаковые веса** (weight sharing)
- На выходе получаем два вектора-эмбеддинга
- Вычисляем евклидово расстояние между ними
- Преобразуем расстояние в процент схожести

### Функция потерь (Contrastive Loss)

```
Loss = label * distance² + (1 - label) * max(0, margin - distance)²
```

Где:
- `label = 1` для одинаковых подписей (минимизируем расстояние)
- `label = 0` для разных подписей (увеличиваем расстояние до margin)
- `margin = 1.0` (порог разделения)

### Аугментация данных

Для улучшения обобщающей способности применяются:
- Случайный поворот (±10°)
- Случайное смещение и масштабирование
- Gaussian Blur
- Нормализация

## 📊 Оценка качества

После обучения проверьте модель:

```bash
python ai/signature_verifier.py
```

**Метрики:**
- **Similarity**: Процент схожести (0-100%)
- **Distance**: Евклидово расстояние (0-∞, обычно 0-2)
- **Confidence**: 
  - `high`: distance < 0.3
  - `medium`: 0.3 ≤ distance < 0.7
  - `low`: distance ≥ 0.7

**Рекомендуемые пороги:**
- `similarity > 50%`: Подписи совпадают
- `similarity < 50%`: Подписи разные

## ⚠️ Важные замечания

1. **Качество данных важнее количества эпох**: Лучше 100 эпох на качественных данных, чем 500 на плохих.

2. **Баланс классов**: В тренировочной выборке должно быть примерно поровну пар "одинаковые" и "разные" подписи.

3. **Переобучение**: Следите за графиками. Если train_loss падает, а val_loss растёт — уменьшите сложность модели или увеличьте данные.

4. **Воспроизводимость**: Для фиксированного результата установите seed:
   ```python
   torch.manual_seed(42)
   np.random.seed(42)
   ```

5. **GPU ускорение**: При наличии CUDA обучение ускорится в 10-50 раз.

## 🎯 Пример использования в коде

```python
from ai.signature_verifier import get_verifier

# Инициализация верификатора
verifier = get_verifier('models/signature_siamese.pth')

# Сравнение двух файлов
result = verifier.verify_signatures('sig1.png', 'sig2.png')

print(f"Схожесть: {result['similarity']}%")
print(f"Совпадение: {result['is_match']}")
print(f"Уверенность: {result['confidence']}")

# Или из байтов (для веб-формы)
with open('sig1.png', 'rb') as f1, open('sig2.png', 'rb') as f2:
    result = verifier.verify_signatures_from_bytes(f1.read(), f2.read())
```
